# github-green-python

让 GitHub 变绿的小脚本

确保你已经下载 python

需要的 GitHub 东西，这些都可以从 settings 中找到

- GitHub Email
- GitHub 用户名
- GitHub Token（只需要勾选 repo 即可，设置 7 天）
- 新建一个 GitHub 仓库，名字随意，私有，然后需要他的 Clone URL

1. 直接克隆本仓库

```bash
git clone git@github.com:hubei-xhjy/github-green-python.git
```

2. 运行脚本

```bash
python main.py
```

然后输入邮箱，用户名和 Github token 登陆即可
